
# Register-Login App
Simple web application to register and login

# Live Site

# Features
- Login with stored information in database
- Register the information by getting a user to database
- HTML, CSS, EJS, NodeJS
- Amazon RDS

# Objective 
- To study NodeJS and ExpressJS
- Learned how to use Amazon RDS
